用户（love38beyond）的偏好：

1. **文档语言**：文档和操作指南使用中文。
2. **MVP 优先**：先做最小可用版本，后续迭代扩展。不追求一步到位。
3. **小规模场景**：面向 20-50 人培训机构内部考试，不需要大规模集群方案。
4. **腾讯云**：部署环境固定在腾讯云广州地域（ap-guangzhou）。
5. **务实风格**：选型偏好成熟稳定方案（如 Guacamole 而非自研 Web 终端），能用就行，不过度设计。
6. **运维考试场景**：考试内容为 Linux 运维操作，学员需要 Windows 管理机 + Linux 目标服务器。
7. **GitHub**：代码托管在 https://github.com/love38beyond/ExamLabPlatform。
8. **管理服务器规格**：倾向于 2核4G 够用即可，后续可在线升级。

**Why:** 这些偏好影响所有技术决策——优先考虑简单、低成本、易维护的方案。

**How to apply:** 提建议时倾向低成本简单方案，文档使用中文，不引入 Kubernetes 等重方案。
